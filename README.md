# subtitle-transform

App for transforming subtitle files.

## Development ##

Requires [boot](https://github.com/boot-clj/boot)

You can get it with [this script](https://github.com/craigy/env/blob/master/boot-clj.sh).

### Development

```bash
boot dev
```

```bash
firefox http://localhost:3000 &
```

## Production ##

```bash
boot prod
```
